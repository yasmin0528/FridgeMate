# FridgeMate
页面一：首页冰箱食材页 
首页用于展示用户数字冰箱中的食材库存信息。系统显示全部食材列表，包括名称和剩余数量。用户可通过翻转卡片查看食材详细信息，包括保质期、库存状态等。用户可选择多个食材加入当前组合，系统记录用户所选食材，用于后续菜谱推荐。翻转卡片支持鼠标悬停（桌面端）或点击（移动端）触发。

页面二：拍照上传页 
拍照上传页用于录入现实冰箱中的食材信息。用户可选择拍照或从相册上传冰箱图片。上传后系统进入识别状态，显示识别进度。识别完成后显示食材识别结果，用户可对结果进行修改、删除或补充。确认后，识别内容自动同步至首页数字冰箱库存。

页面三：食谱教程页 
用户在首页选择多个食材后，系统根据当前食材组合与数据库中的菜谱进行匹配，并推荐最相似的菜谱结果。推荐结果显示菜谱名称、预计时间和基础信息。用户点击菜谱进入教程页面，查看对应教程内容。教程支持跳转外部视频链接或帖子教程链接进行学习。学习完成后返回系统，点击“已完成菜肴”按钮，系统记录一次做饭行为，并更新打卡与经验数据。

页面四：个人成就页 
个人成就页用于记录用户成长信息和使用数据。页面显示用户头像、昵称和经验值信息。经验条根据用户完成做饭次数实时更新。页面下方显示日历打卡记录，系统自动记录用户每日完成做饭任务情况，用户可查看历史打卡数据和连续完成记录
